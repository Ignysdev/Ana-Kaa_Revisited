# Añã-Kaa
Projeto originalmente desenvolvido no IFBA para a disciplina de oficina III, com o objetivo de estimular o debate acerca da invasão de terras indígenas, a preservação da identidade de um povo e o impacto das ações humanas na natureza. Agora atualizado para versões mais recentes e portado para computadores.
